## Local Development

To run Jekyll live-reload server locally:

1. Run `bash run_server.sh` to start the Jekyll live-reload server.
2. Open [http://127.0.0.1:4000](http://127.0.0.1:4000) in your browser.
3. The server will automatically recompile and refresh the page when you modify the source code.
